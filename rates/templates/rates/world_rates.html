Rate Recommendation Process

  1. Data is downloaded from the Inequality-adjusted Human Development Index , which is calculated by the United Nations Development Programme.
  2. Relevant data is then formatted. {% if perms.rates.delete_raterequest %} (see "latest" tab here) {% endif %}
  3. One of six zones (A-F) is assigned to each country. {% if perms.rates.delete_raterequest %} (see "distribution" tab here) {% endif %}
  4. Base rates are established for a list of skill titles. {% if perms.rates.delete_raterequest %} (see "template" tab here){% endif %}
  5. These formatted data are uploaded to the web application and used as inputs for a proprietary algorithm that translates the base rate for each skill into a country-specific pay rate range recommendation (here).
